# 24 点

> 寻找 24 点的最佳解法。

```sh
$ g++ ./24.cpp -Wall -std=c++17 -O3 -o ./24
$ ./24
```

包含了最佳解法的 `24.txt` 和 `24.md` 将会被生成在同一目录下。
